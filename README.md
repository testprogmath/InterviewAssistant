# Интервью Ассистент

Локальное macOS-приложение для записи и анализа интервью с кандидатами.

Записывает твой голос и голос кандидата из системного звука (Zoom, Meet,
Telemost и т.п.) в **две отдельные дорожки**, расшифровывает обе через
WhisperKit на Apple Neural Engine, и отдаёт результат LLM-провайдеру
(DeepSeek / OpenAI / OpenRouter / в перспективе Ollama) для генерации
саммари, SWOT-анализа, рекомендации по найму и уточняющих вопросов.

Все данные хранятся локально. API-ключи лежат в Keychain.

---

## Что умеет

- 🎙 **Запись интервью** — одновременно микрофон ноутбука + системный
  звук через ScreenCaptureKit. С AEC (эхоподавлением) и минимальным
  audio ducking, чтобы голос кандидата не приглушался.
- 📥 **Импорт файла** — можно скинуть готовый аудио или видео файл (mp3,
  m4a, mp4, mov, wav, ...), он автоматически конвертируется и
  расшифровывается. Для одной дорожки разделение спикеров недоступно.
- 📝 **Транскрипция** — WhisperKit `large-v3-turbo`, квантизированный
  под ANE. Русский язык — отлично. Скорость на M1 Pro: ~5–10 мин на
  60-минутное интервью.
- 🧠 **AI-анализ через LLM** — четыре готовых анализа:
  - Саммари (общее впечатление, сильные стороны, опасения, цитаты с
    таймкодами)
  - SWOT-анализ
  - Рекомендация по найму (6 градаций) + обоснование + уверенность
  - 3–5 уточняющих вопросов с темой и обоснованием
- 📋 **История сессий** — sidebar со всеми прошлыми интервью, бейджи
  по доступным артефактам
- 💾 **Сохранение** — каждое интервью лежит на диске как папка с
  `session.json` + аудио в `audio/`. JSON структурированный — потом
  можно искать, экспортировать, анализировать программно.
- 📤 **Экспорт** — копирование в буфер обмена или сохранение в `.md`
  с полным транскриптом + всеми анализами

## Архитектура

```
┌────────────────────────────────────────────────────────────┐
│  UI (SwiftUI)                                              │
│  ContentView · HistoryView · SettingsView                  │
└──────────────────────────┬─────────────────────────────────┘
                           │
                ┌──────────▼──────────┐
                │ InterviewCoordinator│  orchestrator
                └──┬────────┬────┬────┘
                   │        │    │
        ┌──────────▼──┐  ┌──▼─┐ ┌▼──────────────┐
        │AudioCapture │  │... │ │AnalysisProvider│  protocol
        │  Service    │  │    │ │                │
        └─────────────┘  └────┘ └───┬────────────┘
                                    │
                       ┌────────────┼────────────┐
                       ▼            ▼            ▼
                OpenAICompat   OllamaProvider  ...future
                (DeepSeek,     (planned)
                 OpenAI,
                 OpenRouter)
```

- **`Domain/`** — чистые модели (`Session`, `Transcript`, `Summary`,
  `SWOTAnalysis`, `Recommendation`, ...). Всё `Codable`, всё в JSON.
- **`Storage/SessionStore`** — CRUD сессий на файловой системе.
  Структура папки сессии: `audio/` (источники), `transcripts/`
  (промежуточные результаты), `analyses/` (LLM-артефакты), `exports/`
  (генерируемые отчёты).
- **`Services/InterviewCoordinator`** — оркестратор всего пайплайна:
  старт записи → остановка → автоматическая транскрипция → merge
  по таймстемпам → сохранение → LLM-анализы по требованию.
- **`Services/TranscriptionService`** — WhisperKit-обёртка с
  прогрессом, отменой и автоматической загрузкой модели при первом
  запуске.
- **`Services/SpeakerMergeService`** — склейка двух дорожек в одну
  ленту по таймкодам. Никакой настоящей diarisation не нужно — спикеры
  уже разделены на уровне записи.
- **`Services/AnalysisProvider`** — протокол. Любой LLM-провайдер
  реализует один и тот же интерфейс (структурированные JSON-методы +
  стриминговые для чата).

## Установка

### Сборка из исходников

1. Установить Xcode (App Store, ~10 ГБ)
2. Клонировать репо:
   ```bash
   git clone git@github.com:testprogmath/InterviewAssistant.git
   cd InterviewAssistant
   ```
3. Открыть `InterviewAssistant.xcodeproj`
4. В Xcode: **Project → Target InterviewAssistant → Signing &
   Capabilities → Team** → выбрать свой Apple ID (Personal Team
   подойдёт)
5. **Cmd+B** — собрать. При первом запуске WhisperKit скачает модель
   `large-v3-turbo` (~600 МБ), это разовая операция.
6. Запустить через **Products → Show in Finder → двойной клик по
   InterviewAssistant.app** (или Cmd+R если у тебя нет проблем с
   pty в Xcode).

### Передать готовый .app на другой Mac

1. У себя: **Product → Archive → Distribute App → Copy App**
2. Передать `.app` (AirDrop, USB)
3. На другом Mac положить в `/Applications`
4. Снять quarantine:
   ```bash
   xattr -cr /Applications/InterviewAssistant.app
   ```
5. Запустить через ПКМ → Open (только первый раз).

## Настройка LLM-провайдера

При первом запуске анализ работать не будет — нужно ввести API-ключ.
Открой настройки (**Cmd+,**) и выбери провайдер.

### DeepSeek (рекомендуется для РФ)

- Регистрация на https://platform.deepseek.com — обычный email
- Регистрация без карты, $5 пробных кредитов
- Создать API-ключ на странице **API Keys**, скопировать
- Вставить в **Settings → API-ключ → Сохранить**
- Нажать **Проверить соединение** — должно вернуть «DeepSeek: OK»

**Стоимость:** полный анализ 60-минутного интервью (саммари + SWOT +
рекомендация + follow-ups) — около **$0.02**. $5 хватит на ~250
интервью.

### OpenAI / OpenRouter

Доступны через тот же экран настроек. Понадобится VPN если ты в РФ.

### Ollama (планируется)

Локальная альтернатива для оффлайн-работы. Будет позже.

## Требования

- **macOS 15 Sequoia** или новее (нужен для `AVAssetExportSession.export(to:as:)`,
  `AVAudioVoiceProcessingOtherAudioDuckingConfiguration` и `SettingsLink`)
- **Apple Silicon** (M1/M2/M3) для ANE-ускорения WhisperKit
- **~2 ГБ свободного места** для модели Whisper + кеша Core ML

---

## Разрешения

При первом запуске macOS попросит три разрешения:

1. **Микрофон** — записать твой голос
2. **Запись экрана** — захватить системный звук (через ScreenCaptureKit)
3. **Сетевые подключения** — для LLM-провайдеров (если выбраны
   облачные)

Все три необходимы для основной функциональности.

## Где хранятся данные

```
~/Library/Containers/<bundle-id>.InterviewAssistant/Data/
└── Documents/
    ├── Sessions/
    │   └── <UUID>/
    │       ├── session.json    ← полная сессия как JSON
    │       └── audio/
    │           ├── interviewer.caf
    │           └── candidate.caf
    └── huggingface/
        └── models/argmaxinc/whisperkit-coreml/...  ← WhisperKit
```

API-ключи лежат в системном **Keychain** (`com.anna.interview.providers`).

## Технологии

- **SwiftUI** + Combine
- **AVFoundation** — захват микрофона
- **ScreenCaptureKit** — захват системного звука
- **WhisperKit** by Argmax — речь-в-текст на ANE
- **DeepSeek / OpenAI / OpenRouter** — LLM для анализа
- **Keychain Services** — хранение секретов
